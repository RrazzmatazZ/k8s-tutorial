# 获取k8s集群node节点
```
kubectl get pods -A
```

# 查看node节点信息
一个节点的状态包含以下信息:

地址（Addresses）
状况（Condition）
容量与可分配（Capacity）
信息（Info）
```
kubectl describe node ${nodeName}
```
